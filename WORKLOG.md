#### 6. Тооцооллын бүхий л томъёо

**Төвлөрсөн projectCalculations.js модулийн тооцоолол:**

| Талбар | Томъёо | Хэзээ |
|--------|--------|-------|
| BaseAmount | WosHour × 12,500 | Үргэлж |
| TeamBounty | WosHour × 22,500 | Үргэлж |
| NonEngineerBounty | NonEngineerWorkHour × 5,000 | Үргэлж |
| HourPerformance | (RealHour / PlannedHour) × 100 | PlannedHour > 0 |
| EngineerHand | BaseAmount × (200 - HourPerformance) / 100 | PlannedHour > 0 |
| IncomeHR | (WosHour + additionalHour) × 110,000 | Үргэлж |
| ProfitHR | IncomeHR - (EngineerHand + NonEngineerBounty + additionalValue) | Үргэлж |

**Time Attendance aggregation:**
- RealHour = Sum of (workingHour + overtimeHour)
- EngineerWorkHour = Sum where Position='Инженер'
- NonEngineerWorkHour = Sum where Position≠'Инженер'

#### 8. Firestore Field Names Constants - Code Quality сайжруулалт

**Асуудал:**

- Талбарын нэр солигдох үед олон газар засах шаардлагатай
- Баримтжуулалт байхгүй бол бүр хүмүүс талбарын нэрийг санахгүй

**Б. Ашиглалт жишээ:**
```javascript
// ❌ Өмнө: Алдаа гарч болох
```
